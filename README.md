easy-mode:

* clone with --recursive
* run update.bat (just because)
* run build.bat

requires visual studio 2022 with c++ and cmake + winrar or 7zip

pro-version:

* clone with --recursive
* extract vendor/nyx/vendor/v8/lib/v8_monolith_x64-release
* build with VS2022 in release mode (you can do debug if you like to try)
* install
* copy scripts/ to out/install/x64-release/bin
* run simple_injector (install)
* read the readme
